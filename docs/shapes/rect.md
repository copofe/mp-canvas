# 矩形

| Property | Description              | Type                       | Required | Default |
| -------- | ------------------------ | -------------------------- | -------- | ------- |
| rx       | Horizontal border radius | `number`                   | false    | 0       |
| ry       | Vertical border radius   | `number`                   | false    | 0       |
| radii    | border radius            | `number` &#124; `number[]` | false    | 0       |
